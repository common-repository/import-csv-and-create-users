== Description ==

Import users from a csv files and send a custom message using specific shortcodes.

You can choose capabilities for new users.

(original import csv script from AndyDunn)
== Installation ==

1. Upload `import-users-csv.php` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. You will see a new 'Import CSV' option under the existing 'Users' menu area.
